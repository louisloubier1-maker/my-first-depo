# Shoez Store Project

Ce projet a été réalisé à partir d’un prototype Figma.  
L’objectif était de reproduire une maquette figma simple en HTML et CSS.

## Utilisation de l’IA

J’ai utilisé une IA (chatgpt) comme outil pendant le projet.  
Elle m’a surtout servi pour :
- M'expliquer la synthaxe de html
- mieux organiser mon code,
- corriger certains détails HTML/CSS et certaines erreurs
- m'expliquer comment on utilise certaines fonctions, leurs sens ect.
- trouver une structure simple pour la page,


Je suis resté responsable de mon projet, du choix des éléments à intégrer et du rendu final m'étant basé sur mon projet figma. 

L’IA m’a aidé à avancer plus vite et à mieux comprendre certaines parties du code car en deux semaines c'était compliqué de comprendre un langage de programmation de A à Z. 

## Exemples de demandes faites à l’IA

Voici quelques exemples de choses que j’ai pu demander :

- comment reproduire une page Figma en HTML/CSS simple,
- comment organiser une page avec un header, une colonne de filtres et des cartes produits,
- comment corriger l’alignement des éléments dans le footer,
- comment faire en sorte que certaines parties aient la même police et la même taille,
- Peux-tu me corriger mon code car j'ai une erreur lors de l'execution.
- Aide moi à Aligner les deux headers car je n'y arrive pas, voici mon code : ... . 

## Organisation des fichiers

- `index.html` : structure de la page
- `style.css` : mise en forme
- `assets/` : images utilisées sur le site
- `README.md` : fichier expliquant l'utilisation de l'IA.
